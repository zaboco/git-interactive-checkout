# Git Interactive Checkout

Checkout git branches interactively

## Usage

Install the package globally:
```sh
npm install -g git-interactive-checkout
```

Inside a git project, run the CLI app:
```sh
git-interactive-checkout
```

Or the shorthand:
```sh
gic
```
